FORMULÁRIO CONCEITO

A) Na tag input com atributo type="text", para que serve o atributo "placeholder"?

O atributo especifica uma pequena dica que descreve o valor esperado de um campo de entrada
(por exemplo, um valor amostral ou uma breve descrição do formato esperado) placeholder A dica curta é exibida no campo de entrada antes que o usuário digite um valor. Nota: O atributo funciona com os seguintes tipos de entrada: texto, pesquisa, url, tel, e-mail e senha.placeholder


B) Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Os conjuntos de propriedades verificados ou retornam o estado verificado de um botão de rádio. Esta propriedade reflete o atributo verificado HTML.


C) Na tag input para que serve o atributo "value"?

O atributo especifica o valor de um elemento.value input O atributo é usado de forma diferente para diferentes tipos de entrada:value Para "botão", "reset" e "enviar" - define o texto no botão Para "texto", "senha" e "oculto" - ele define o valor inicial (padrão) do campo de entrada Para "caixa de seleção", "rádio", "imagem" - define o valor associado à entrada (este também é o valor enviado no envio) Nota: O atributo não pode ser usado com .value input type="file"


D) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

O rádio Button é um elemento do formulário que nos permite uma opção entre um conjunto de possibilidades.
É inserido através da tag "input type=”radio”". Com o atributo name damos um nome para agrupar os rádios buttons, para que somente se possa escolher uma opção entre várias outras


E) Para que serve a tag label? Para que serve o atributo "for" nessa tag?

Representa uma legenda para um item em uma interface de usuário. O atributo “for” serve para referenciar o id da tag input, deve conter o mesmo nome.


F) Na tag option para que serve o atributo "selected"?

O atributo é um atributo booleano.selected Quando presente, ele especifica que uma opção deve ser pré-selecionada quando a página é carregada. A opção pré-selecionada será exibida primeiro na lista suspensa. Ponta: O atributo também pode ser definido após o carregamento da página, com um JavaScript.selected


G) Nas tags de formulário para que serve o atributo "required"?

O atributo é um atributo booleano.required Quando presente, especifica que o elemento deve ser preenchido antes de enviar o formulário.


H) Na tag form para que serve o atributo "action"?

O atributo especifica para onde enviar os dados do formulário quando um formulário é enviado.action.


I) Na tag form para que serve o atributo "method"?

O atributo especifica como enviar dados de formulário (o formulário-dados é enviado para a página especificada no atributo).method action Os formulários podem ser enviados como variáveis de URL (com) ou como transação pós HTTP (com). method="get"method="post"


J) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

POST: Anexa dados de formulário dentro do corpo da solicitação HTTP (os dados não são mostrados estão na URL); Não tem limitações de tamanho; Submissões de formulários com POST não podem ser marcadas. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos? O valor a ser informado deve ser o "post", pois ele não tem limitações de tamanho e pode ser usado para enviar grandes quantidades de dados.
